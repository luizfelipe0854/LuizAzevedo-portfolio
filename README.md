# LuizAzevedo-portfolio

Portfólio pessoal desenvolvido com React + Vite + Tailwind CSS.

## Tecnologias

- React 19.1.0
- Vite 5.4.10
- Tailwind CSS v4
- React Icons

## Estrutura do Projeto

- **Componentes Atômicos**
- **Sistema de Cores Customizado**
- **Padrão Atomic Design**

## Como executar

```bash
npm install
npm run dev
```
